## Context

Расширение хранит флаг `showMilliseconds` в `chrome.storage.local` (или `sync`). Он читается при инициализации `content.js` и в `popup.js`. Функция `formatVideoTime(seconds, showMs)` использует этот флаг — обновление UI плеера происходит на тикере `requestAnimationFrame` / `setInterval`, поэтому смена значения в памяти автоматически отразится в отображаемом времени.

Кнопка копирования времени (`.ytp-copy-time-btn`) вставляется после `.ytp-time-display` в control bar — это проверенный паттерн, к которому привязывается новая кнопка.

## Goals / Non-Goals

**Goals:**
- Один клик в плеере = переключение отображения миллисекунд.
- Полная синхронизация с popup-чекбоксом через `chrome.storage`.
- Визуально различимые состояния вкл/выкл.

**Non-Goals:**
- Отдельная клавиша-шорткат (можно добавить позже).
- Хранение состояния пер-видео (флаг глобальный).
- Поддержка Shorts.

## Decisions

### 1. Один источник истины — `chrome.storage`

При клике на кнопку content script пишет новое значение в `chrome.storage`. Popup и content script слушают `chrome.storage.onChanged` и обновляют UI. Это исключает рассинхронизацию между двумя точками управления.

### 2. Размещение — после `.ytp-copy-time-btn`

Кнопка вставляется после кнопки копирования времени. Если кнопка копирования отсутствует — fallback после `.ytp-time-display`. Это сохраняет визуальную группу «таймер + действия над временем».

### 3. Визуальные состояния через CSS-модификатор

```css
.ytp-ms-toggle-btn { opacity: 1; }
.ytp-ms-toggle-btn--off { opacity: 0.45; }
```

Иконка одна, состояние различается через прозрачность/контур. Альтернатива — две разные иконки — отклонена: больше графических ассетов, выше когнитивная нагрузка.

### 4. Иконка — текстовая `.ms` или SVG

Решение: SVG (часы с тремя цифрами) — консистентно с уже используемым SVG в кнопке копирования. Текст `.ms` выглядел бы инородно среди иконок плеера.

## Risks / Trade-offs

- **Гонка `storage.onChanged` ↔ локальное состояние** → Смягчение: всегда читать актуальное значение из storage в обработчике, не кэшировать в IIFE-переменной.
- **Кнопка дублирует функцию popup** → Принимается: это два разных пути для разных контекстов использования.
- **Селектор `.ytp-copy-time-btn` — внутренний для расширения** → Допустимо: он гарантированно добавляется тем же расширением.
